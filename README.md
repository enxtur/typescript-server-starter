### Getting started

Run the following command on your local environment:

```shell
git clone --depth=1 https://github.com/enxtur/actix-server-starter.git my-project-name
cd my-project-name
cargo build
```

Then, you can start locally in development mode

```shell
cargoo run
```

### Testing
